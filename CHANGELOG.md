# Changelog

All notable changes to this project will be documented in this file.

## 1.0.2 (2026-04-17)

### ✨ Features

- **DraggableList**: 增加拖拽列表组件，支持项的勾选启用与拖拽排序功能。

## 1.0.1 (2026-03-05)

### ✨ Features

- **Performance**: 增加性能监控组件 `PerformanceDebugger` 并调整全局样式。
- **Loading**: 优化滚动加载组件 `LoadingByScroll` 的交互逻辑。

## 1.0.0 (2026-03-04)

### ✨ Features

- **Initial Release**: `lstDesign` 组件库初始版本发布。
- **Core Components**: 包含 `DraggableModal`, `CustomRangePicker`, `LoadingByScroll` 等核心组件。
